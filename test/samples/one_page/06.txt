<h1 class="devsite-page-title" style="width:50%" id="compose-material-3-adaptive" data-text="Compose Material 3 Adaptive" tabindex="-1">Compose Material 3 Adaptive</h1>




  



  






  




<div style="align:right; text-align:right; padding-bottom:21px; margin-top:-45px; border-bottom:1px solid black;"> 
&nbsp;&nbsp;
</div>



<div style="float:left; font-size:medium; margin-bottom:5px; margin-top:10px" class="nocontent">
<div style="text-align:left; font-size:90%; padding-right:30px; margin-right:10px; margin-bottom:10px;">

<a href="/reference/kotlin/androidx/compose/material3/adaptive/package-summary">androidx.compose.material3.adaptive</a><br>

<div style="text-align:left; padding-left:10px; margin-left:10px; margin-top:15px; margin-bottom:10px;">


Use the Material 3 adaptive library to create adaptive UIs
that will adapt themselves automatically according to the current window
configurations like window size classes or device postures. The library provides
both default scaffold implementations and necessary building block composables
to create your own custom experiences.
